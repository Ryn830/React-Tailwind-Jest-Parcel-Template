## React + Tailwind + Jest + Parcel Template

### Create your new repo using the GitHub CLI

Install [gh](https://cli.github.com/)

`brew install gh`

then:

`gh repo create <NEW_REPO_NAME> --template Ryn830/React-Tailwind-Jest-Parcel-Template`

### Install

`yarn`

### Develop

`yarn start`

### Test

`yarn test`
